Georeferencer Extended Crosshairs

This plugin toggles full-window red crosshairs in the QGIS Georeferencer window to assist with precise GCP placement.

Usage

  • Toggle via the Red Crosshair Icon on the main toolbar.
  • Toggle via the Plugins > Georeferencer Extended Crosshairs menu.
  • Toggle via the hotkey: Ctrl+Alt+X.
